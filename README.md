slops is a simple unix-like operating system, written in assembly and C.

## BUILD ISO IMAGE

### 1. build

```bash
make
```

## BOOT THE SYSTEM

```bash
make run
```

## REMOVE ARTIFACTS

```bash
make clean
```

or if you want to remove all generated files run:

```bash
make distclean
```
